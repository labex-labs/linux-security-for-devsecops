A hands-on Linux security course for DevSecOps learners focused on service exposure, web hardening, file permissions, sudo, secrets, root-run automation, service identity, runtime configuration, logs, scripts, and access policy cleanup. You will investigate real host state, apply targeted fixes, and verify that systems stay usable after hardening.
